================================================================================
ERROR DETAILS
================================================================================

Error Message:
"'block' tag with name 'extra_js' appears more than once"

Location:
templates/hr_dashboard.html, line 956

Cause:
- Duplicate {% block extra_js %} declaration
- First block at line 711
- Second block at line 956
- Django doesn't allow duplicate block names

================================================================================
WHAT WAS WRONG
================================================================================

Template Structure (BEFORE):
----------------------------
{% block content %}
    ... content ...
{% endblock %}

{% block extra_js %}
    <script>
        ... Chart.js and other scripts ...
    </script>
{% endblock %}

{% block extra_js %}  ← DUPLICATE! This caused the error
    <script>
        ... Attendance edit scripts ...
    </script>
{% endblock %}

================================================================================
WHAT WAS FIXED
================================================================================

Template Structure (AFTER):
---------------------------
{% block content %}
    ... content ...
{% endblock %}

{% block extra_js %}
    <script src="https://cdn.jsdelivr.net/npm/chart.js"></script>
    <script>
        ... Chart.js scripts ...
        ... Attendance edit scripts ...  ← Merged into one block
    </script>
{% endblock %}

Changes Made:
-------------
1. Removed duplicate {% block extra_js %} declaration
2. Merged attendance edit scripts into the existing extra_js block
3. Kept all functionality intact
4. Maintained proper block structure
