FEATURE 1: ROLE MANAGEMENT SYSTEM
================================================================================

WHAT WAS ADDED:
---------------
1. Role field in EmployeeProfile model with 4 roles:
   - Employee (Regular User)
   - Team Leader (Manage Team)
   - HR (Full HR Access)
   - Admin (Full System Access)

2. Role Management UI in Master Data List:
   - New "Change Role" button (user-tag icon)
   - Shows current role as badge under status
   - Modal dialog for changing roles
   - Permission descriptions for each role

3. Backend View:
   - change_employee_role() function
   - Validates role selection
   - Updates is_hr flag automatically
   - Logs all role changes in audit trail

4. Audit Logging:
   - New action type: 'role_change'
   - Tracks who changed role, when, and what changed
   - Full audit trail for compliance

HOW TO USE:
-----------

5. Select new role from dropdown

7. Role updated and logged!

FEATURE 2: ATTENDANCE STATUS EDITING
================================================================================

WHAT WAS ADDED:
---------------
1. New fields in Attendance model:
   - status_modified_by (who edited)
   - status_modified_at (when edited)
   - status_change_reason (why edited)

2. Edit Status UI in HR Dashboard:
   - New "Edit" button (yellow) in attendance table
   - Shows "Edited" indicator if status was modified
   - Hover shows who edited and when
   - Modal dialog for editing status

3. Backend View:
   - edit_attendance_status() function
   - Validates status selection
   - Requires reason for change
   - Logs all changes in audit trail

4. Audit Logging:
   - New action type: 'attendance_status_change'
   - Tracks old status → new status
   - Records reason for change
   - Full audit trail

HOW TO USE:
-----------
1. Login as HR/Admin
2. Go to HR Dashboard: http://127.0.0.1:8000/attendance/hr/
3. Click "Attendance" tab
4. Find employee attendance record
5. Click yellow "Edit" button
6. Select new status (Present/Late/Half-Day/Absent)
7. Enter reason for change (REQUIRED)
8. Click "Update Status"
9. Status updated and logged!

STATUS OPTIONS:
---------------
- Present (Full Day) - 7.5+ hours
- Late - Arrived late but worked full day
- Half-Day - Less than 7.5 hours
- Absent - Did not attend

AUDIT TRAIL:
------------
All changes are logged with:
- Who made the change
- When it was changed
- Old status → New status
- Reason for change
- Employee affected

View audit logs in HR Dashboard → Audit Logs tab
